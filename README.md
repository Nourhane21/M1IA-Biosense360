# M1IA-Biosense360
Des capteurs environnementaux collectent en continu des données (la température, l’humidité, la qualité de l’air) et des indices de confort thermique. L’objectif du projet est de transformer ces mesures brutes en informations directement exploitables pour évaluer l’habitabilité d’un lieu  favorable ou à risque pour les humains et pour les plantes.
